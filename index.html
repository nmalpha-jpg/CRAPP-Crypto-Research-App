GoPlus Security API
Paste any ERC-20 / BEP-20 contract address to instantly check for honeypot, hidden mint functions, owner privileges, tax settings, and other rug pull indicators.
🚩 Common Red Flags
  • ⛔ Honeypot — can buy but not sell
  • ⛔ High buy/sell tax (>10%)
  • ⛔ Owner can mint unlimited tokens
  • ⛔ Trading can be paused by owner
  • ⛔ Proxy contract (logic can change)
  • ⛔ External calls in transfer function
✅ Green Flags
  • ✅ Verified open-source contract
  • ✅ Renounced ownership
  • ✅ LP tokens locked
  • ✅ Low/zero buy-sell tax
  • ✅ No mint function
  • ✅ Audited by known firm
